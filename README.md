## Library Usage

# Installation

Add the dependency:

```bash
npm i react-native-search-header
```

## Peer Dependencies

<h5><i>IMPORTANT! You need install them</i></h5>

```js
"react": ">= 16.x.x",
"react-native": ">= 0.55.x",
"react-native-androw": ">= 0.0.34"
```

# Usage

## Import

```jsx
import { SearchHeader } from "react-native-search-header";
```

## Fundamental Usage

```jsx
<SearchHeader />
```
